Computational Geometry - KD Tree

Purpose

This project was created for Computational Geometry course conducted by Ph.D. Barbara Głut at AGH University of Science and Technology.

Presentation

The project was first presented in the classroom. Here is a link to the the slides (in polish language): http://slid.es/mequrel/kddrzewa

Code

Application was created as a webapp in order to be accessible and useful to anyone. I used generic web application skeleton provided by "yo" app generator. The functional code is placed in scripts/ directory and divided on 5 modules:

  • binarytree.js - displays binary tree visualizer, highly influenced by: ...
  • chart.js - displays geometrical representation of kd-tree, highly influenced by: ...
  • history.js - contains descriptor service implementation and is able to present algorithm history dumped during its execution
  • kdtree.js - core part, algorithms implementation
  • main.js - combines all modules together, has view state machine implemented

Algorithm history visualization is implemented using the idea of execution log. Each algorithm has injected special service called describer. Using this service algorithm is able to describe each step both textually and graphically. Here's an example taken from search function performing range query: 

describer.nextStep()
    .write("Finished recursion for left/up part.")
    .setCurrentNode(kdTree)
    .highlightRange(kdTree.left.range);
Using describer algorithm can merge many describing request in one step. Fluent syntax interface of the describer helps with code readability.

Acknowledgments and reference

I took big advantage by using d3.js library along with their brilliant examples. Two of those examples: Collapsible tree layout and Quadtree were adapted to visualize kd-tree algorithms. Thanks mbostock!

Application was created using yeoman workflow. As previously described app skeleton was generated using yo tool. The app was compiled, minified and deployed using grunt. Besides d3.js also jQuery and highlight.js were used.
